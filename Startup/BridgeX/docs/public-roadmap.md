# Public Roadmap

## Phase 1 — MVP: Manual Investment Flow
- Stablecoin funding request intake
- Basic investor profile capture
- Manual investment request submission and status tracking

## Phase 2 — Investor Dashboard
- Portfolio summary view (instrument, amount, maturity)
- Basic transaction history
- Downloadable statement/receipt

## Phase 3 — Operations Tooling
- Internal views for managing request status
- Better visibility into pending/active/completed investments

## Phase 4 — Future Improvements
- More automation where partners and regulation allow
- Additional supported investment products and rails
